# Federal Data

Place FCC and NTIA bulk downloads here.

## Expected Files (v0.3.0)

- `bdc_broadband_map_*.csv` — FCC Broadband Data Collection (served/unserved BSLs)
- `funding_map_*.csv` — FCC Broadband Funding Map (all federal awards by census block)
- `bead_allocations.csv` — NTIA BEAD state allocations and approval status
- `erate_commitments_*.csv` — USAC E-Rate funding commitments
