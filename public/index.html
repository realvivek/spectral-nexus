Scoring Methodology

Opportunity Score (0-100)

Each county receives a composite score based on six weighted factors:

  • Coverage Gap (25%) — % of BSLs that are underserved or unserved
  • Unserved % (20%) — % of BSLs with zero qualifying broadband
  • Funding Availability (20%) — BEAD status × unserved concentration
  • Income Need (15%) — Lower income = higher grant eligibility
  • Population Density (10%) — Sweet spot: 50-200/sq mi (buildable, not saturated)
  • 5G Readiness (10%) — Tower density, fiber backhaul, spectrum availability

Data Sources

  • Demographics: US Census ACS 5-Year Estimates
  • BSL / Coverage: FCC Broadband Data Collection (BDC)
  • Funding: NTIA BEAD Dashboard, FCC Auction 904 (RDOF)
  • Infrastructure: FCC ASR (tower locations)

Disclaimer

This is a demonstration tool. Telecom metrics (BSLs, coverage, fiber, towers) use realistic simulated data for the current 200-county dataset. As we integrate real FCC BDC bulk data, mock values will be replaced with verified figures. Scoring methodology is transparent and open to feedback.
